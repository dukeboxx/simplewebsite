What is a commit?

In git, a commit just refers to any time you edit, add or delete a file. All commits are logged so that everyone else who has access to the workflow know exactly what has been changed.

What is a branch?

The term branch refers to a separate version of your code that you can make changes and test things without editing the main version of your project.

What is a pull request?

A pull request is a way of submitting your work to the others working on the same proect for review and to get added to the main branch.

Additional information

Once a pull request has been submitted, you can deploy another branch to ensure everything works as intended before merging it with the main branch and creating your final product.


The Unix Shell

The Unix shell is a command line interface (CLI) and a scripting language. This means that it allows users to directly input commands that the computer can interpret and execute.

How to list the files in the directory

To list the files in a directory, you simply have to enter 'ls' in the Unix command prompt.

How to show and change the working directory

The working directory just refers to the directory you are currently in. To show the working directory you use the command 'pwd' in the Unix prompt. This stands for print current working directory
To change the working directory, you use the 'cd' command followed by the name of the directory you wish to change. To navigate to the home directory you can use 'cd ..'.

How to create a new directory

To create a new directory you use the command 'mkdir' followed by whatever you want to call this new directory.

How to create and remove files

To create a new file, you use the command 'touch'.
To delete a file, you use the command 'rm'.
